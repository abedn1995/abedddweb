RINGKAS : Mempercepat Kinerja Windows 10

By Abedn.Nov 29, 2021

3 Cara Mempercepat Kinerja Windows 10

Mempercepat kinerja Windows 10
adalah untuk membuat windows kita bekerja agar lebih ringan dan tidak
lemot dalam melakukan pengoprasian program yang dijalankan.

Baik langkah 1 : cara matikan Background Apps

Buka Windows kemudian klik Pengaturan/Setting

Langkah ke  2 : Kemudian buka klik Privacy (Location, Camera).

Langkah ke 3 : Selanjutnya klik Background Apps kemudian silahkan Off kan App yang tidak perlu seperti Get Office dan yang lainnya.

Oke itu tadi Cara Pertama untuk Mempercepat Kinerja Windows 10 kamu.

Cara Kedua Mengurangi Effect Animation

Langkah ke 1 : Buka File Folder ( Windows  + E ) Kemudian Klik kanan This PC setelah itu pilih Properties.

Langkah ke 2 : Silahkan klik Advanced system settings , lalu klik settings pada bagian Performance. kemudian hilangkan Ceklis/ tanda centang seperti ini.

Langkah ke 3 : Silahkan kamu simpan Apply kemudian klik OK.
Itulah Cara kedua untuk Mempercepat Kenerja Windows 10 kamu.

Cara Ketiga Uninstal Programs Yang tidak Perlu

Langkah ke 1 : Buka menu Tampilan Cepat ( Windows + X ) Setelan muncul Menu tampilan Cepat lalu Pilih Programs and Features. 

Langkah ke 2 : Silahkan pilih Software yang kamu rasa tidak di butuhkan dan tidak perlu.

Langkah ke 3 : Kemudian klik software lalu Klik Uninstal atau juga bisa dengan Double klik ( klik 2 x software ) kemudian Yes.
